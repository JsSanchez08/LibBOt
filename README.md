# LibBOt

Cliente de escritorio para Windows que permite buscar y descargar libros desde Library Genesis y Anna's Archive en una sola interfaz.

## Instalación

1. Descarga el `LibBOt-vX.Y.Z.exe` más reciente desde **[Releases](../../releases)**.
2. Doble clic. Listo.

No requiere Python ni dependencias. La primera ejecución crea:

- `%APPDATA%\LibBOt\libbot.db` — favoritos, historial y ajustes.
- `%USERPROFILE%\Downloads\LibBOt\` — carpeta de descargas (cambiable en Ajustes).

## Características

- Búsqueda en Libgen y Anna's Archive.
- Filtros por formato, idioma y año.
- Paginación de 10 resultados por página.
- Favoritos e historial persistentes.
- Descarga directa con selector manual de mirror opcional.
- Tema claro/oscuro/sistema.
